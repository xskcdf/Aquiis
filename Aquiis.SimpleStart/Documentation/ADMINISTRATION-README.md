# Administration

Components for system administration and user management.

## Components

-   **UserManagement.razor** - User administration interface with phone number management

## Features

-   User listing with comprehensive information
-   Phone number display and management
-   User search functionality
-   Last login tracking
-   Role and permission management
-   Admin-only access controls

## Navigation

Access via the main navigation menu "Administration" link (admin users only).

## Permissions

**Admin Only** - All components in this directory require administrative privileges.

## Security

These components contain sensitive user information and system controls.
Access is restricted through authorization policies.
