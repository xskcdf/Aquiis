# Properties Management

Components for managing rental properties in the system.

## Components

-   **Properties.razor** - Main properties listing with search, sort, and pagination
-   **CreateProperty.razor** - Form for adding new properties (if implemented)
-   **EditProperty.razor** - Form for editing existing properties (if implemented)
-   **ViewProperty.razor** - Detailed property view (if implemented)

## Features

-   Property listing with professional grid interface
-   Sortable columns (address, type, rent amount, status)
-   Pagination with configurable page sizes
-   Search functionality
-   Responsive design with Bootstrap styling

## Navigation

Access via the main navigation menu "Properties" link.

## Permissions

All authenticated users can view properties.
Admin users can create, edit, and manage properties.
