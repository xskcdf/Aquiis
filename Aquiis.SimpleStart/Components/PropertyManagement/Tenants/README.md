# Tenants Management

Components for managing tenant information in the system.

## Components

-   **Tenants.razor** - Main tenants listing with search, sort, and pagination
-   **CreateTenant.razor** - Form for adding new tenants (if implemented)
-   **EditTenant.razor** - Form for editing existing tenants (if implemented)
-   **ViewTenant.razor** - Detailed tenant view (if implemented)

## Features

-   Tenant listing with professional grid interface
-   Sortable columns (name, email, phone, status)
-   Pagination with configurable page sizes
-   Search functionality
-   Contact information management
-   Responsive design with Bootstrap styling

## Navigation

Access via the main navigation menu "Tenants" link.

## Permissions

All authenticated users can view tenants.
Admin users can create, edit, and manage tenant information.
